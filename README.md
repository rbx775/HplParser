## HPL Parser
Blender to HPL3 Engine Parser for Mod Creation. ❤️ Frictional Games

### Supported Blender Versions
- [x] 3.6+
- [x] 4.0+

### Supported Games
- [x] Amnesia The Bunker
- [ ] Amnesia Rebirth
